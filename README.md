## DBファイル読み込み手順  
1. <http://localhost/phpmyadmin>からphpMyAdminへ
2. 作成したdbを選択し、上部タブの_インポート_をクリック
3. アップロードファイルとしてdbファイル(create_db.sql)を選択し、他の項目はそのままで実行
